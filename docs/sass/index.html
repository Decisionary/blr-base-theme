    @function contrast-color($color, $dark: $color-dark, $light: $color-light) { ... }
    View source

    Description

    Takes a color, determines whether it's dark or light, and returns a color that contrasts it. For example, if the color is dark, a light color is returned, and vice-versa. Useful for when you need to ensure an element's text is readable against its background color.
